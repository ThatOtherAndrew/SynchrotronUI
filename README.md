# Vite Svelte Flow Template

This template creates a very basic [Svelte Flow](https://svelteflow.dev) app with [Vite](https://vite.dev).

## Get it!

```sh
npx degit xyflow/vite-svelte-flow-template app-name
```

## Installation

```sh
npm install
```

## Development

```sh
npm run dev
```

## Build

```sh
npm run build
```
